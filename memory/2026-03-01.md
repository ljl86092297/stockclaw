# 2026-03-01 - 可视化股票软件策略优化

## 📊 项目概述
为用户李金雷的A股可视化股票软件开发高胜率短线交易策略，目标是实现≥90%胜率的"主升浪狙击·价托回踩"策略。

## 🏗️ 系统架构分析
### 技术栈
- **前端**: Vue 3 + Element Plus + Lightweight Charts
- **后端**: FastAPI + SQLAlchemy + Pandas
- **数据源**: Baostock + Akshare
- **数据库**: SQLite

### 核心功能模块
1. **数据获取层** (`data_source.py`): Baostock获取A股K线数据，支持增量更新
2. **策略引擎** (`backtest_engine.py`): 基础回测引擎
3. **信号检测** (`useSignalDetection.js`): 前端JavaScript实时计算信号
4. **数据存储**: stocks表、price_data表、scan_records表

### 策略类型
1. **V1: 弹簧突破·基础版**
2. **V2: 弹簧突破·进阶版**
3. **V3: 主升浪狙击·价托回踩** ⭐（当前优化目标）

## 🎯 V3策略核心逻辑
### 四个步骤
1. **强势蓄势**: 涨停基因 + 均线价托形成
2. **回踩确认**: 5日均线偏离≤阈值
3. **买入执行**: T+1高开买入
4. **盈利判定**: 日内涨幅目标

### 关键参数
- 涨停基因: N日内M次涨停≥X%
- 均线价托: 5/10/20日均线多头排列
- 回踩确认: 偏离≤Y%
- 买入执行: 高开A%-B%
- 盈利判定: 日内涨C%-D%

## 🔧 技术挑战与解决方案
### 遇到的问题
1. **浏览器自动化失败**: Chrome扩展连接断开
2. **后端API未响应**: 可能服务未运行
3. **参数过于严格**: 初始参数无信号生成

### 解决方案
1. **创建自动化脚本**: 直接通过API测试策略
2. **参数优化算法**: 多组合测试找到最佳平衡
3. **批量测试框架**: 50只非9开头股票测试

## 🚀 创建的自动化工具
### 文件列表
1. **`strategy_optimizer.py`** - 完整策略优化器（12.8KB）
   - 参数组合搜索
   - 胜率计算模型
   - 结果保存功能

2. **`strategy_tester.py`** - 简化版测试器（7.3KB）
   - 3种参数组合测试
   - 模拟胜率计算
   - 报告生成

3. **`run_strategy_optimization.bat`** - 一键运行批处理（0.6KB）
   - 自动检查环境
   - 安装依赖
   - 运行优化

### 参数优化组合
1. **组合1（宽松）**: 20日涨停≥8.0%，偏离≤0.05，高开0.0-6.0%
2. **组合2（平衡）**: 15日涨停≥8.5%，偏离≤0.03，高开0.5-4.5%
3. **组合3（严格）**: 10日涨停≥9.5%，偏离≤0.02，高开1.0-3.0%

## 📈 优化方法论
### 迭代优化流程
1. **初始测试**: 宽松参数获取初始信号
2. **模式分析**: 成功/失败案例对比
3. **参数调整**: 针对性优化关键参数
4. **验证测试**: 新参数批量验证

### 关键洞察
1. **涨停基因**: 放宽天数和幅度可显著增加信号量
2. **回踩确认**: 偏离阈值是胜率关键控制点
3. **高开范围**: 影响买入机会和风险控制平衡
