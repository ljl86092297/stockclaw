# AI产业链完整链条研究框架

## 一、产业链全景图

### 1. 电力基础设施层
- **发电端**：火电、水电、核电、风电、光伏、储能
- **输电端**：特高压、智能电网、电力设备
- **配电端**：配电自动化、微电网、电力交易

### 2. 硬件基础设施层
- **芯片制造**：
  - 设计：EDA工具、IP核
  - 制造：晶圆代工、封装测试
  - 材料：硅片、光刻胶、特种气体
- **服务器与数据中心**：
  - 服务器：CPU、GPU、AI加速卡
  - 存储：内存、SSD、硬盘
  - 网络：交换机、路由器、光模块
- **终端设备**：
  - 智能终端：手机、PC、IoT设备
  - 边缘计算：边缘服务器、网关

### 3. AI核心技术层
- **基础算法**：机器学习、深度学习、强化学习
- **大模型**：语言模型、多模态模型、行业模型
- **开发框架**：TensorFlow、PyTorch、PaddlePaddle
- **工具链**：训练平台、推理引擎、部署工具

### 4. AI应用层
- **行业解决方案**：
  - 金融：智能投顾、风控、反欺诈
  - 医疗：辅助诊断、药物研发、健康管理
  - 制造：智能质检、预测维护、工艺优化
  - 交通：自动驾驶、智能调度、车路协同
- **产品与服务**：
  - 软件：SaaS应用、行业软件
  - 硬件：机器人、智能设备
  - 平台：云平台、生态平台

## 二、市值筛选标准

### 筛选条件：
1. 市值低于500亿元人民币
2. 在产业链关键环节有核心技术
3. 财务健康，有持续盈利能力
4. 研发投入占比较高
5. 行业地位稳固，有竞争优势

### 排除标准：
1. ST/*ST公司
2. 连续亏损企业
3. 主营业务不清晰
4. 技术依赖外部

## 三、研究方法

### 数据来源：
1. 东方财富、同花顺等财经平台
2. 公司年报、招股说明书
3. 行业研究报告
4. 专利数据库

### 分析维度：
1. 技术实力：专利数量、研发团队、技术壁垒
2. 市场地位：市场份额、客户质量、品牌影响力
3. 财务健康：营收增长、利润率、现金流
4. 成长潜力：行业空间、技术趋势、政策支持
