## 1. 风险管理的核心原则

### 1.1 短线交易的风险特点
1. **时间短** → 风险暴露时间有限
2. **频率高** → 需要更严格的风险控制
3. **胜率优先** → 宁可错过，不可做错
4. **亏损有限** → 单笔最大亏损必须控制

### 1.2 风险管理目标
```
短期目标：单笔最大亏损 < 账户资金的2%
中期目标：月度最大回撤 < 10%
长期目标：年度最大回撤 < 20%
```

## 2. 仓位管理策略

### 2.1 凯利公式（简化版）
```
f = p - (1-p)/b
其中：
f = 理论最优仓位
p = 胜率
b = 赔率（平均盈利/平均亏损）

实际应用：
理论仓位 = f
实际仓位 = 理论仓位 × 0.5（保守调整）
```

### 2.2 具体仓位规则
```
胜率预估 > 70%：仓位 20-25%
胜率预估 60-70%：仓位 15-20%
胜率预估 50-60%：仓位 10-15%
胜率预估 < 50%：不交易

总仓位限制：账户资金的30%
单只股票仓位：总仓位的1/3到1/2
```

### 2.3 动态仓位调整
1. **盈利加仓**：达到第一目标后，可加仓10%
2. **亏损减仓**：连续2笔亏损，仓位减半
3. **市场环境**：市场不好时，总仓位降至15%

## 3. 止损策略

### 3.1 止损类型
1. **技术止损**：
   - 跌破关键支撑位（如MA20）
   - 趋势改变（如ADX转弱）
   - 指标转坏（如RSI超买后下跌）

2. **百分比止损**：
   - 低风险交易：最大亏损4%
   - 中风险交易：最大亏损6%
   - 高风险交易：最大亏损8%

3. **时间止损**：
   - 持有3天无盈利：考虑退出
   - 持有7天未达目标：强制退出
   - 持有14天：无论如何都退出

### 3.2 止损位置计算
```
买入价：¥100
止损百分比：6%
止损价：¥100 × (1 - 0.06) = ¥94

或取技术止损和百分比止损的较严格者
```

### 3.3 移动止损
```
初始止损：买入价 × (1 - 6%)
价格上涨10%后：当前价 × (1 - 4%)
价格上涨20%后：当前价 × (1 - 3%)
永远不让盈利变成亏损
```

## 4. 止盈策略

### 4.1 止盈目标设定
```
基于风险收益比：
目标收益 = 止损幅度 × 风险收益比
要求：风险收益比 > 2.0

例如：
止损6% → 目标收益12%（风险收益比2.0）
止损5% → 目标收益15%（风险收益比3.0）
```

### 4.2 分批止盈
```
买入仓位：20%
第一目标（10%）：卖出10%（一半仓位）
第二目标（15%）：卖出5%（剩余一半）
最后5%：移动止损持有
```

### 4.3 动态止盈
1. **趋势跟踪**：趋势未破，继续持有
2. **指标确认**：RSI>80考虑止盈
3. **成交量**：放量滞涨考虑止盈

## 5. 风险收益比管理

### 5.1 风险收益比计算
```
风险收益比 = 目标收益 / 止损幅度

要求：
最低：2.0
良好：3.0
优秀：4.0+
```

### 5.2 不同风险收益比的策略
```
风险收益比 < 2.0：放弃交易
风险收益比 2.0-3.0：标准仓位
风险收益比 > 3.0：可适当增加仓位
```

### 5.3 胜率与风险收益比的关系
```
高胜率(>70%)：可接受较低风险收益比(2.0)
中胜率(60-70%)：需要较高风险收益比(2.5-3.0)
低胜率(<60%)：需要很高风险收益比(>3.5)
```

## 6. 资金曲线管理

### 6.1 资金曲线保护
1. **连续亏损控制**：
   - 连续2笔亏损：仓位减半
   - 连续3笔亏损：停止交易1天
   - 连续4笔亏损：停止交易3天，复盘策略

2. **最大回撤控制**：
   - 回撤5%：检查策略
   - 回撤10%：降低仓位
   - 回撤15%：停止交易，全面复盘

### 6.2 盈利保护
1. **盈利回撤控制**：
   - 盈利回撤20%：止盈一半
   - 盈利回撤30%：全部止盈
   - 盈利回撤50%：强制止盈

2. **盈利加仓规则**：
   - 账户盈利10%：可增加总仓位5%
   - 账户盈利20%：可增加总仓位10%
   - 账户盈利30%：保持仓位，提取部分利润

## 7. 心理风险管理

### 7.1 常见心理陷阱
1. **恐惧**：该买不买，该卖不卖
2. **贪婪**：不止盈，想赚更多
3. **希望**：不止损，希望反弹
4. **后悔**：追涨杀跌，频繁交易

### 7.2 应对方法
1. **计划交易**：提前制定买卖计划
2. **严格执行**：不因情绪改变计划
3. **记录分析**：每笔交易记录心理状态
4. **定期复盘**：分析心理错误，改进

## 8. 系统集成

### 8.1 风险管理系统架构
```
输入：交易信号 + 风险评估
处理：
  1. 仓位计算（基于胜率和风险收益比）
  2. 止损计算（技术和百分比取严格者）
  3. 止盈计算（基于风险收益比）
  4. 动态调整（移动止损、分批止盈）
输出：完整的交易计划
```

### 8.2 代码实现要点
```python
class RiskManager:
    def calculate_position(self, win_rate, risk_reward):
        """计算仓位"""
        # 凯利公式简化版
        kelly = win_rate/100 - (1-win_rate/100)/risk_reward
        # 保守调整
        position = max(0.05, min(0.25, kelly * 0.5))
        return position
    
    def calculate_stop_loss(self, buy_price, technical_stop, max_loss_pct):
        """计算止损价"""
        pct_stop = buy_price * (1 - max_loss_pct/100)
        # 取较严格者
        return min(technical_stop, pct_stop)
    
    def calculate_take_profit(self, buy_price, stop_loss, risk_reward):
        """计算止盈价"""
        loss_amount = buy_price - stop_loss
        target_profit = loss_amount * risk_reward
        return buy_price + target_profit
```
